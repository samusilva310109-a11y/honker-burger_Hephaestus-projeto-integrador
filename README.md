## Orientações

1. **FORK** - Apenas **1 Integrante deve realizar o Fork**
2. Adicionar todos os integrantes como colaboradores no repositório
3. Utilize esta estrutura para manter seu projeto organizado e facilitar o trabalho colaborativo dos colegas.

## Sobre este Projeto

O projeto **Honker Burger - Hephaestus** consiste no desenvolvimento de uma loja virtual para divulgação dos produtos da hamburgueria Honker Burger, integrada a uma área administrativa para gerenciamento de produtos e categorias. 
A aplicação contará com Landing Page, catálogo de produtos, organização por categorias, página de detalhes dos produtos e uma área administrativa responsável pelo gerenciamento dos dados da plataforma.
